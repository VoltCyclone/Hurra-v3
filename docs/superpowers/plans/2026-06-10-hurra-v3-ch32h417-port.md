# Hurra-v3 CH32H417 Port Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Port the Hurra-v2 USB HID man-in-the-middle firmware (NXP i.MX RT1062 / Cortex-M7) to the WCH CH32H417 dual-core RISC-V MCU, capturing the real HID device on USBHS (host) and presenting the clone to the PC on USBFS (device), with the latency-critical relay on the V5F core and command/humanize/LED/telemetry offload on the V3F core.

**Architecture:** Preserve Hurra-v2's hardware-agnostic logic layer (`hurra.c`/`ferrum.c`/`proto.h`/`actions.c`/`humanize.c`/`desc_capture` logic) behind stable public headers; rewrite the hardware layer (`usb_host.c`→USBHS host, `usb_device.c`→USBFS device, `kmbox.c` UART backend→USART+DMA, `led.c`→GPIO/TIM, `main.c`→two core entries) against WCH registers; add `icc.c` (inter-core channel: lock-free SPSC rings in shared SRAM + HSEM rendezvous + IPC doorbell). V3F is the boot/master core: it sets clocks then releases V5F.

**Tech Stack:** RISC-V GCC 12 (`riscv-none-elf-gcc`, `-march=rv32imac -mabi=ilp32`), WCH StdPeriph-style EVT library (vendored under `vendor/wch`), WCH-LinkE + WCH OpenOCD, TinyFrame (Hurra protocol), host-native C for unit tests.

**Reference documents (READ FIRST):**
- Spec: `docs/superpowers/specs/2026-06-10-hurra-v3-ch32h417-port-design.md`
- **Register/boot reference: `docs/ch32h417-evt-reference.md`** — cites exact EVT SDK files for every register and API used below. When a task says "see EVT ref §N", read that section.
- Hurra-v2 source to port from: `../Hurra-v2/src/`, `../Hurra-v2/test/`, `../Hurra-v2/tools/`.

**Conventions for every task:**
- Work in `/Users/ramseymcgrath/code/Hurra-v3`.
- ISRs use `__attribute__((interrupt("WCH-Interrupt-fast")))` and the vector-table handler names (external = `*_IRQHandler`, core/IPC/HSEM = `*_Handler`).
- USB DMA buffers: USBFS needs the `+0x20000000` CPU-side offset via `USBFSD_UEP_BUF(N)`; USBHS does NOT. Never put USB buffers in code-ITCM.

- [ ] **Step 3: Add a USB-DMA-safe buffer section to the V5F linker script**

USB buffers must live in DTCM/SRAM the USB DMA engine can reach, NOT in code-ITCM. Add to `core/link_v5f.ld` inside `SECTIONS`, placed in `RAM` (DTCM), before `.bss`:

```ld
  .usbdma (NOLOAD) :
  {
    . = ALIGN(512);
    PROVIDE(_usbdma_start = .);
    *(.usbdma)
    *(.usbdma.*)
    . = ALIGN(4);
    PROVIDE(_usbdma_end = .);
  } >RAM
```

> Buffers are tagged `__attribute__((section(".usbdma"), aligned(4)))` in `usb_host.c`/`usb_device.c`. The `+0x20000000` USBFS translation is applied at access time via `USBFSD_UEP_BUF`, independent of placement.

- [ ] **Step 2: Write `src/led.c` (GPIO via StdPeriph; heartbeat via TIM2)**

See EVT ref §6. `led_blink_forever` uses `Delay_Ms` (from vendored Debug). The heartbeat uses TIM2 update IRQ to toggle at the configured rate.

```c
#include "ch32h417_port.h"
#include "board.h"
#include "led.h"
#include "debug.h"   // Delay_Ms / Delay_Init

static volatile uint16_t s_centihz = 100;
static volatile uint32_t s_tim_reload;

void led_init(void)
{
    RCC_HB2PeriphClockCmd(LED_RCC_HB2, ENABLE);
    GPIO_InitTypeDef g = {0};
    g.GPIO_Pin   = LED_GPIO_PIN;
    g.GPIO_Speed = GPIO_Speed_Very_High;
    g.GPIO_Mode  = GPIO_Mode_Out_PP;
    GPIO_Init(LED_GPIO_PORT, &g);
}

void led_on(void)     { GPIO_SetBits(LED_GPIO_PORT, LED_GPIO_PIN); }
void led_off(void)    { GPIO_ResetBits(LED_GPIO_PORT, LED_GPIO_PIN); }
void led_toggle(void) { GPIO_WriteBit(LED_GPIO_PORT, LED_GPIO_PIN,
                          (BitAction)!GPIO_ReadOutputDataBit(LED_GPIO_PORT, LED_GPIO_PIN)); }

void led_blink_forever(uint8_t count, uint16_t on_ms, uint16_t off_ms)
{
    for (;;) {
        for (uint8_t i = 0; i < count; i++) {
            led_on();  Delay_Ms(on_ms);
            led_off(); Delay_Ms(off_ms);
        }
        Delay_Ms(400);
    }
}

// Heartbeat: TIM2 update IRQ toggles the LED. Reload computed from centihz.
// TIM2 on HB1 bus, clocked at V3F HCLK (~100 MHz). period_ticks = HCLK / (psc+1) / freq_hz.
static void heartbeat_apply_rate(uint16_t centihz)
{
    if (centihz == 0) centihz = 100;
    // toggles per second = 2 * (centihz/100). Use PSC=9999 -> 10 kHz tick base.
    uint32_t tick_hz = BOARD_V3F_HZ / 10000u;            // ~10 kHz
    uint32_t toggles_per_sec = (2u * centihz + 50u) / 100u;
    if (toggles_per_sec == 0) toggles_per_sec = 1;
    s_tim_reload = tick_hz / toggles_per_sec;
    TIM2->ATRLR = (s_tim_reload ? s_tim_reload - 1u : 0u);
}

void led_heartbeat_start(void)
{
    RCC_HB1PeriphClockCmd(RCC_HB1Periph_TIM2, ENABLE);
    TIM_TimeBaseInitTypeDef t = {0};
    t.TIM_Prescaler     = 9999;                 // 10 kHz tick base
    t.TIM_CounterMode   = TIM_CounterMode_Up;
    t.TIM_Period        = 4999;                 // overwritten by apply_rate
    t.TIM_ClockDivision = TIM_CKD_DIV1;
    TIM_TimeBaseInit(TIM2, &t);
    heartbeat_apply_rate(s_centihz);
    TIM_ClearITPendingBit(TIM2, TIM_IT_Update);
    NVIC_SetPriority(TIM2_IRQn, 16);
    NVIC_EnableIRQ(TIM2_IRQn);
    TIM_ITConfig(TIM2, TIM_IT_Update, ENABLE);
    TIM_Cmd(TIM2, ENABLE);
}

void led_heartbeat_set_rate(uint16_t centihz)
{
    s_centihz = centihz;
    heartbeat_apply_rate(centihz);
}

void TIM2_IRQHandler(void) __attribute__((interrupt("WCH-Interrupt-fast")));
void TIM2_IRQHandler(void)
{
    if (TIM_GetITStatus(TIM2, TIM_IT_Update) == SET) {
        led_toggle();
        TIM_ClearITPendingBit(TIM2, TIM_IT_Update);
    }
}
```

> Register/field names (`ATRLR`, `GPIO_ReadOutputDataBit`, `TIM_TimeBaseInit`) are StdPeriph; verify exact spellings against `vendor/wch/Peripheral/inc/ch32h417_tim.h` and `_gpio.h` while implementing. `led.c` is a V3F-only module (heartbeat IRQ runs on V3F).

- [ ] **Step 1: Write `src/main_v3f.c` (master core)**

```c
// main_v3f.c — V3F is the boot/master core: clocks, then release V5F.
#include "ch32h417_port.h"
#include "board.h"
#include "led.h"
#include "debug.h"

int main(void)
{
    SystemInit();                 // V3F sets the PLL/clock tree (EVT ref §2)
    SystemAndCoreClockUpdate();
    Delay_Init();

    // Release V5F: set its PC to its flash origin and wake it.
    NVIC_WakeUp_V5F(Core_V5F_StartAddr);   // 0x00010000 (EVT ref §1)

    // HSEM rendezvous: V3F enables the semaphore IRQ; V5F will take+release it.
    HSEM_ITConfig(HSEM_ID0, ENABLE);

    led_init();
    for (;;) {
        led_toggle();
        Delay_Ms(500);            // slow blink = V3F alive
    }
}
```

- [ ] **Step 2: Write `src/main_v5f.c` (slave core, temporary blink)**

```c
// main_v5f.c — V5F slave entry. Phase 1: prove it was released by V3F.
#include "ch32h417_port.h"
#include "board.h"
#include "led.h"
#include "debug.h"

int main(void)
{
    SystemAndCoreClockUpdate();   // V5F does NOT call SystemInit (EVT ref §2)
    Delay_Init();

    HSEM_FastTake(HSEM_ID0);      // rendezvous with V3F (EVT ref §1)
    HSEM_ReleaseOneSem(HSEM_ID0, 0);

    // Temporary: fast blink proves V5F is running. Shares the LED with V3F for
    // bring-up only; Phase 5 replaces this with the USB relay loop and the LED
    // is owned by V3F. To disambiguate now, V5F drives the LED faster.
    for (;;) {
        led_on();  Delay_Ms(80);
        led_off(); Delay_Ms(80);
    }
}
```

> Bring-up caveat: both cores poking one LED is only to confirm both run. If contention makes the result ambiguous on hardware, temporarily point V5F at a second GPIO pin. The Phase-2 ICC counter test is the rigorous proof; this step is a smoke test.

- [ ] **Step 3: Write the `Makefile` (two images + merge + flash)**

```makefile
# Hurra-v3 — CH32H417 dual-core. V3F master + V5F slave, merged to one image.
TOOLCHAIN ?= riscv-none-elf
CC      = $(TOOLCHAIN)-gcc
OBJCOPY = $(TOOLCHAIN)-objcopy
SIZE    = $(TOOLCHAIN)-size

# Match EVT SDK: RV32IMAC + soft-float ilp32 (EVT ref toolchain note).
# zicsr is REQUIRED: the WCH startup .S uses csrw/csrr (mstatus/mtvec/mepc + WCH
# custom CSRs 0xBC0/0xBC1/0x804). GCC 12 split CSR ops into the zicsr extension,
# so bare rv32imac fails to assemble startup. Verified with riscv64-unknown-elf.
ARCH = -march=rv32imac_zicsr -mabi=ilp32

# Toolchain note: the plan was written for riscv-none-elf-gcc (MounRiver). This
# environment has homebrew's riscv64-unknown-elf-gcc, which builds the same
# rv32imac/ilp32 target. Build with: make TOOLCHAIN=riscv64-unknown-elf

# Protocol selector: 'hurra' (binary, TinyFrame, default) or 'ferrum' (ASCII).
PROTOCOL ?= hurra
ifeq ($(PROTOCOL),hurra)
  CMD_BAUD  ?= 4000000
  PROTO_DEF  = -DPROTOCOL_HURRA
  PROTO_SRC  = src/hurra.c src/third_party/TinyFrame/TinyFrame.c
else ifeq ($(PROTOCOL),ferrum)
  CMD_BAUD  ?= 115200
  PROTO_DEF  = -DPROTOCOL_FERRUM
  PROTO_SRC  = src/ferrum.c
else
```

- [ ] **Step 1: Write `src/icc.h` — fixed 16-byte records, two SPSC rings**

```c
#pragma once
#include <stdint.h>
#include <stdbool.h>

// Inter-core channel. Two single-producer/single-consumer rings in shared SRAM.
// V3F->V5F: injection commands. V5F->V3F: telemetry. Lock-free; HSEM only for
// the one-time init rendezvous. IPC channel 0 is the V3F->V5F doorbell.

#define ICC_RECORD_BYTES   16
#define ICC_RING_SLOTS     256          // power of two
#define ICC_RING_MASK      (ICC_RING_SLOTS - 1)

// Record tags (tag in byte 0).
enum {
    ICC_TAG_NONE = 0,
    // V3F -> V5F (inject commands)
    ICC_TAG_INJECT_MOUSE,
    ICC_TAG_INJECT_KEYBOARD,
    ICC_TAG_CLICK_RELEASE,
    ICC_TAG_KB_RELEASE,
    ICC_TAG_SET_BAUD,
    ICC_TAG_SET_HUMAN_LEVEL,
    ICC_TAG_PHYS_MASK,
    // V5F -> V3F (telemetry)
    ICC_TAG_TELEM_COUNTS,
    ICC_TAG_TELEM_STATUS,
    // Phase-2 smoke test
    ICC_TAG_PING,
    ICC_TAG_PONG,
};

typedef struct {
    uint8_t tag;
    uint8_t b[ICC_RECORD_BYTES - 1];    // payload, tag-specific
} icc_record_t;

typedef struct {
    volatile uint32_t head;             // producer writes
    volatile uint32_t tail;             // consumer writes
    icc_record_t slot[ICC_RING_SLOTS];
} icc_ring_t;

// The shared block lives at a fixed address in both images' .shared section.
typedef struct {
    volatile uint32_t magic;            // set by V3F at init
    icc_ring_t v3f_to_v5f;
    icc_ring_t v5f_to_v3f;
} icc_shared_t;

#define ICC_MAGIC  0x48563343u          // 'HV3C'

void icc_init_v3f(void);   // master: zero rings, set magic, then signal via HSEM
void icc_init_v5f(void);   // slave: wait for magic via HSEM rendezvous

// Producer side (returns false if ring full).
bool icc_send_to_v5f(const icc_record_t *r);   // call from V3F
bool icc_send_to_v3f(const icc_record_t *r);   // call from V5F

// Consumer side (returns false if empty).
bool icc_recv_from_v3f(icc_record_t *out);     // call from V5F
bool icc_recv_from_v5f(icc_record_t *out);     // call from V3F

// Doorbell: V3F rings V5F after enqueue so V5F can wfi when idle.
void icc_ring_doorbell_v5f(void);              // V3F side
// V5F's IPC_CH0_Handler clears the doorbell; defined in icc.c.
```

- [ ] **Step 2: Write `src/icc.c`**

The shared block is placed in `.shared` so both linker scripts map it to `0x20178000`. RISC-V `fence` provides the release/acquire ordering for the SPSC indices.

```c
#include "ch32h417_port.h"
#include "icc.h"

// Single instance, pinned into shared SRAM (0x20178000) by both linker scripts.
__attribute__((section(".shared_data")))
icc_shared_t g_icc;

static inline void mem_fence(void) { __asm volatile("fence" ::: "memory"); }

void icc_init_v3f(void)
{
    for (uint32_t i = 0; i < ICC_RING_SLOTS; i++) {
        g_icc.v3f_to_v5f.slot[i].tag = ICC_TAG_NONE;
        g_icc.v5f_to_v3f.slot[i].tag = ICC_TAG_NONE;
    }
    g_icc.v3f_to_v5f.head = g_icc.v3f_to_v5f.tail = 0;
    g_icc.v5f_to_v3f.head = g_icc.v5f_to_v3f.tail = 0;
    mem_fence();
    g_icc.magic = ICC_MAGIC;
    mem_fence();
}

void icc_init_v5f(void)
{
    // V5F waits until V3F has initialized the shared block.
    while (g_icc.magic != ICC_MAGIC) { __asm volatile("nop"); }
    mem_fence();
}

static bool ring_push(icc_ring_t *r, const icc_record_t *rec)
{
    uint32_t head = r->head;
    uint32_t next = (head + 1) & ICC_RING_MASK;
    if (next == (r->tail & ICC_RING_MASK)) return false;   // full
    r->slot[head & ICC_RING_MASK] = *rec;
    mem_fence();                                           // record before head
    r->head = next;
    return true;
}

static bool ring_pop(icc_ring_t *r, icc_record_t *out)
{
    uint32_t tail = r->tail;
    if ((tail & ICC_RING_MASK) == (r->head & ICC_RING_MASK)) return false; // empty
    *out = r->slot[tail & ICC_RING_MASK];
    mem_fence();                                           // record before tail
    r->tail = (tail + 1) & ICC_RING_MASK;
    return true;
}

bool icc_send_to_v5f(const icc_record_t *r)  { return ring_push(&g_icc.v3f_to_v5f, r); }
bool icc_send_to_v3f(const icc_record_t *r)  { return ring_push(&g_icc.v5f_to_v3f, r); }
bool icc_recv_from_v3f(icc_record_t *out)    { return ring_pop(&g_icc.v3f_to_v5f, out); }
bool icc_recv_from_v5f(icc_record_t *out)    { return ring_pop(&g_icc.v5f_to_v3f, out); }

void icc_ring_doorbell_v5f(void)
{
    // IPC channel 0, bit0: V3F sets the flag; V5F gets IPC_CH0_Handler.
    IPC_SetFlagStatus(IPC_CH0, IPC_CH_Sta_Bit0);
}

// V5F-side doorbell handler: just clears the flag; the relay loop drains the ring.
void IPC_CH0_Handler(void) __attribute__((interrupt("WCH-Interrupt-fast")));
void IPC_CH0_Handler(void)
{
    if (IPC_GetITStatus(IPC_CH0, IPC_CH_Sta_Bit0)) {
        IPC_ClearFlagStatus(IPC_CH0, IPC_CH_Sta_Bit0);
    }
}
```

> Verify `IPC_SetFlagStatus`/`IPC_GetITStatus`/`IPC_ClearFlagStatus` argument enums against `vendor/wch/Peripheral/inc/ch32h417_ipc.h` (EVT ref §1). The `.shared_data` section name must match what the linker `KEEP`s.

```c
            }
        }

    }
```

See EVT ref §5. RX uses a circular DMA channel into a ring; the read cursor chases the DMA write position via `DMA_GetCurrDataCounter`. TX uses a software ring drained into a DMA channel on completion.

```c
#include "ch32h417_port.h"
#include "board.h"
#include "uart.h"

#define RX_RING_SZ  1024
#define TX_RING_SZ  4096

static uint8_t  rx_dma[RX_RING_SZ];
static uint16_t rx_read_pos;
static uint32_t rx_total;

static uint8_t  tx_ring[TX_RING_SZ];
static volatile uint16_t tx_head, tx_tail;
static volatile bool     tx_dma_busy;
static uint32_t tx_total;
static uint32_t s_baud;
static uint32_t err_or, err_fe, err_ne;

// DMA channels for the chosen USART (USART2: TX=DMA1_Ch7, RX=DMA1_Ch6 per EVT §5).
#define RX_DMA_CH   DMA1_Channel6
#define TX_DMA_CH   DMA1_Channel7
#define TX_DMA_IRQn DMA1_Channel7_IRQn

static void rx_dma_setup(void)
{
    DMA_InitTypeDef d = {0};
    DMA_DeInit(RX_DMA_CH);
    d.DMA_PeripheralBaseAddr = (uint32_t)CMD_USART_DATAR;
    d.DMA_MemoryBaseAddr     = (uint32_t)rx_dma;
    d.DMA_DIR                = DMA_DIR_PeripheralSRC;
    d.DMA_BufferSize         = RX_RING_SZ;
    d.DMA_PeripheralInc      = DMA_PeripheralInc_Disable;
    d.DMA_MemoryInc          = DMA_MemoryInc_Enable;
    d.DMA_PeripheralDataSize = DMA_PeripheralDataSize_Byte;
    d.DMA_MemoryDataSize     = DMA_MemoryDataSize_Byte;
    d.DMA_Mode               = DMA_Mode_Circular;
    d.DMA_Priority           = DMA_Priority_VeryHigh;
    d.DMA_M2M                = DMA_M2M_Disable;
    DMA_Init(RX_DMA_CH, &d);
    DMA_Cmd(RX_DMA_CH, ENABLE);
    rx_read_pos = 0;
}

static void tx_dma_kick(void)
{
    if (tx_dma_busy) return;
    uint16_t head = tx_head, tail = tx_tail;
    if (head == tail) return;                 // nothing to send
    uint16_t chunk = (head > tail) ? (head - tail)
                                   : (uint16_t)(TX_RING_SZ - tail); // to ring end
    DMA_InitTypeDef d = {0};
    DMA_DeInit(TX_DMA_CH);
    d.DMA_PeripheralBaseAddr = (uint32_t)CMD_USART_DATAR;
    d.DMA_MemoryBaseAddr     = (uint32_t)&tx_ring[tail];
    d.DMA_DIR                = DMA_DIR_PeripheralDST;
    d.DMA_BufferSize         = chunk;
    d.DMA_PeripheralInc      = DMA_PeripheralInc_Disable;
    d.DMA_MemoryInc          = DMA_MemoryInc_Enable;
    d.DMA_PeripheralDataSize = DMA_PeripheralDataSize_Byte;
    d.DMA_MemoryDataSize     = DMA_MemoryDataSize_Byte;
    d.DMA_Mode               = DMA_Mode_Normal;
    d.DMA_Priority           = DMA_Priority_High;
    d.DMA_M2M                = DMA_M2M_Disable;
    DMA_Init(TX_DMA_CH, &d);
    tx_dma_busy = true;
    DMA_ITConfig(TX_DMA_CH, DMA_IT_TC, ENABLE);
    DMA_Cmd(TX_DMA_CH, ENABLE);
}

void uart_init(uint32_t baud)
{
    s_baud = baud;
    RCC_HB1PeriphClockCmd(CMD_USART_RCC_HB1, ENABLE);
    RCC_HBPeriphClockCmd(RCC_HBPeriph_DMA1, ENABLE);
    // GPIO AF for USART2 TX/RX (PA2/PA3): configure per board; see ch32h417_gpio.h.
    // (StdPeriph: GPIO_Init AF_PP for TX, IN_FLOATING for RX, + remap if needed.)

    USART_InitTypeDef u = {0};
    u.USART_BaudRate   = baud;
    u.USART_WordLength = USART_WordLength_8b;
    u.USART_StopBits   = USART_StopBits_1;
    u.USART_Parity     = USART_Parity_No;
    u.USART_HardwareFlowControl = USART_HardwareFlowControl_None;
    u.USART_Mode       = USART_Mode_Tx | USART_Mode_Rx;
    USART_Init(CMD_USART, &u);
    USART_DMACmd(CMD_USART, USART_DMAReq_Tx | USART_DMAReq_Rx, ENABLE);
    USART_Cmd(CMD_USART, ENABLE);

    rx_dma_setup();
    tx_head = tx_tail = 0; tx_dma_busy = false;
    NVIC_SetPriority(TX_DMA_IRQn, 32);
    NVIC_EnableIRQ(TX_DMA_IRQn);
}

void uart_set_baud(uint32_t baud)

uint16_t uart_rx_available(void)
{
    uint16_t dma_pos = RX_RING_SZ - DMA_GetCurrDataCounter(RX_DMA_CH);
    return (uint16_t)((dma_pos - rx_read_pos) & (RX_RING_SZ - 1));
}

uint16_t uart_rx_read(uint8_t *dst, uint16_t max)
{
    uint16_t n = 0;
    uint16_t dma_pos = RX_RING_SZ - DMA_GetCurrDataCounter(RX_DMA_CH);
    while (rx_read_pos != dma_pos && n < max) {
        dst[n++] = rx_dma[rx_read_pos];
        rx_read_pos = (rx_read_pos + 1) & (RX_RING_SZ - 1);
    }
    rx_total += n;
    return n;
}

uint16_t uart_tx_room(void)
{
    uint16_t used = (uint16_t)((tx_head - tx_tail) & (TX_RING_SZ - 1));
    return (uint16_t)(TX_RING_SZ - 1 - used);
}

uint16_t uart_tx_write(const uint8_t *src, uint16_t len)
{
    uint16_t n = 0;
    while (n < len && uart_tx_room() > 0) {
        tx_ring[tx_head] = src[n++];
        tx_head = (tx_head + 1) & (TX_RING_SZ - 1);
    }
    tx_total += n;
    tx_dma_kick();
    return n;
}

void uart_tx_flush(void) { tx_dma_kick(); }

uint32_t uart_overrun(void)       { return err_or; }
uint32_t uart_framing(void)       { return err_fe; }
uint32_t uart_noise(void)         { return err_ne; }
uint32_t uart_rx_byte_count(void) { return rx_total; }
uint32_t uart_tx_byte_count(void) { return tx_total; }

void DMA1_Channel7_IRQHandler(void) __attribute__((interrupt("WCH-Interrupt-fast")));
void DMA1_Channel7_IRQHandler(void)
{
    if (DMA_GetITStatus(DMA1_IT_TC7)) {
        DMA_ClearITPendingBit(DMA1_IT_TC7);
        DMA_Cmd(TX_DMA_CH, DISABLE);
        tx_tail = (uint16_t)((tx_tail + DMA_GetCurrDataCounter(TX_DMA_CH)) & 0); // see note
        // Correct tail advance: we sent `chunk` = the BufferSize we programmed.
        // Recompute below instead of relying on counter (which reads 0 at TC).
        tx_dma_busy = false;
        tx_dma_kick();   // send next chunk (e.g. ring wrap remainder)
    }
}
```

```c
    USBFSD->UEP1_TX_LEN = len;

        USBFSD->UEP0_TX_CTRL = USBFS_UEP_T_RES_NAK;

    }
```

### Task 5.1: USBHS host driver behind usb_host.h

**Files:**

- [ ] **Step 1: Write `src/usb_host.c` implementing the v2 host API on USBHS**

Implement against EVT ref §4. Core pieces:
- DMA buffers `__attribute__((section(".usbdma"), aligned(4)))` for RX/TX (USBHS: no `+0x20000000`).
- `usb_host_init`: `USBHS_RCC_Init` (480 MHz PLL + UTMI) then host init (`CFG`, `RX/TX_DMA`, `RX_MAX_LEN`, `PORT_CFG=PD_EN|HOST_EN`, `FRAME|=SOF_CNT_EN`, `CFG=SOF_EN|DMA_EN|PHY_SUSPENDM`).
- `usb_host_device_connected`: `USBHSH->PORT_STATUS_CHG & USBHS_UHIF_PORT_CONNECT` (or PORT_STATUS connect bit).
- `usb_host_port_reset`: `USBHSH_ResetRootHubPort` sequence.
- `usb_host_device_speed`: read PORT_STATUS speed bits → `USB_SPEED_HIGH/FULL/LOW`.
- `usb_host_control_transfer`: SETUP token, DATA in/out loop with toggle, STATUS stage — port `USBHSH_CtrlTransfer`. Also the `_fire`/`_async_busy` variants (for v2's async control); implement `_fire` as a non-blocking single-stage kick with a busy flag, or (simplest correct) make `_fire` synchronous and `_async_busy` always false. **Pick synchronous** for first correctness; note the simplification in a comment.
- `usb_host_interrupt_init(slot,addr,ep,maxpkt)`: store per-slot endpoint addr/number/toggle/maxpkt.
- `usb_host_interrupt_poll_zerocopy(slot,&ptr,len)`: issue `USBHSH_GetEndpData` (IN token, timeout 0); on success set `*ptr = USBHS_RX_Buf` and return length; on NAK return 0. (Zero-copy: return the DMA buffer pointer directly, valid until the next poll — matches v2 semantics.)
- `usb_host_interrupt_out_init` / `usb_host_interrupt_out_send`: OUT token to the stored EP; return false if a prior send is still in flight (synchronous impl: always completes, return true/false on ACK).

Single transaction primitive (port `USBHSH_Transact`, EVT ref §4):
```c
static uint8_t usbhs_transact(uint8_t pid_ep, uint16_t tog, uint32_t timeout)
{
    USBHSH->CONTROL = USBHS_UH_HOST_ACTION | (pid_ep & 0x0F) | ((pid_ep & 0xF0) << 0) | tog;
    // NOTE: pid in low nibble, ep in bits[7:4]; build pid_ep = pid | (ep<<4).
    USBHSH->INT_FLAG = USBHS_UHIF_TRANSFER;
    while (timeout-- && !(USBHSH->INT_FLAG & USBHS_UHIF_TRANSFER)) { /* Delay_Us(1) */ }
    return (uint8_t)(USBHSH->INT_ST & USBHS_UH_T_TOKEN_MASK);
}
```

```c

#include "icc.h"
#include <string.h>

static int32_t  inj_dx, inj_dy;
static uint8_t  inj_buttons;
static int8_t   inj_wheel;
static uint8_t  inj_kb_mod, inj_kb_keys[6];

void usb_merge_init(void) { inj_dx = inj_dy = 0; inj_buttons = 0; inj_wheel = 0; }

void usb_merge_drain_icc(void)
{
    icc_record_t r;
    while (icc_recv_from_v3f(&r)) {
        switch (r.tag) {
        case ICC_TAG_INJECT_MOUSE: {
            int16_t dx = (int16_t)(r.b[0] | (r.b[1] << 8));
            int16_t dy = (int16_t)(r.b[2] | (r.b[3] << 8));
            inj_dx += dx; inj_dy += dy;
            inj_buttons = r.b[4];
            inj_wheel  += (int8_t)r.b[5];
        } break;
        case ICC_TAG_INJECT_KEYBOARD:
            inj_kb_mod = r.b[0];
            memcpy(inj_kb_keys, &r.b[1], 6);
            break;
        /* SET_BAUD/SET_HUMAN_LEVEL/PHYS_MASK handled here or ignored on V5F */
        default: break;
        }
    }
}

void usb_merge_report(uint8_t iface_protocol, uint8_t *report, uint8_t len)
{
    if (iface_protocol == 2 && len >= 3) {       // mouse: [buttons, dx, dy, (wheel)]
        report[0] |= inj_buttons;
        int16_t nx = (int8_t)report[1] + clamp_take(&inj_dx);  // clamp to int8, carry
        int16_t ny = (int8_t)report[2] + clamp_take(&inj_dy);
        report[1] = (uint8_t)nx; report[2] = (uint8_t)ny;
        if (len >= 4) report[3] = (uint8_t)((int8_t)report[3] + inj_wheel), inj_wheel = 0;
    } else if (iface_protocol == 1 && len >= 8) { // keyboard: overlay modifier+keys
        report[0] |= inj_kb_mod;
        /* merge injected keys into empty slots — port v2 logic */
    }
}
```
